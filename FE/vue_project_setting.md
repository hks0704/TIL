1. Vue 프로젝트 생성
```
npm init vue@latest my-vue-project
```
2. 체크하는 설정. 나머지 option은 No.
  - TypeScript
  - Router
  - Pinia
  - Vitest
  - ESLint
  - Prettier
3. npm install
```
cd my-vue-project
npm install
```
4. tailwindCSS install
```
npm install -D tailwindcss@3 postcss autoprefixer
npx tailwindcss init -p
```
5. tailwindCSS config 파일에 확장자명 설정 추가
6. 환경변수 설정 추가
```
.env
```
7. 실행
```
npm run format
npm run dev
```
### 기타
- 설치 플러그인
  - vue sniffet
  - vue official
